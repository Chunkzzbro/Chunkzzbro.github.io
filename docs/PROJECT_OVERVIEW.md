# Project Overview — ML Systems Portfolio

## Objective
To transform a standard vCard template into a high-credibility, production-grade engineering portfolio for **Renjith Anil**, specializing in **Machine Learning Systems and Applied AI**.

## Positioning
The portfolio is designed to signal expertise in the "infrastructure" of AI:
- **Distributed Systems:** OpenMPI-based CNN training.
- **Inference Optimization:** Low-latency CV pipelines (~30 FPS).
- **Observability:** Prometheus integration for ML monitoring.
- **Cloud Native:** AWS SageMaker deployment workflows.

## Key Innovation: The Admin Suite
Unlike static templates, this project implements a **client-side CMS**:
- **Authenticated Access:** Secure login for content management.
- **Versioning:** Full edit history with timestamped restore points.
- **Interactive Editing:** Hover-based management of links, images, and metrics without touching code.
- **Persistence:** LocalStorage-based data lifecycle with default fallback.
